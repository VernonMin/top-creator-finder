========================
  Top Creator Finder 项目方案文档
========================

## 项目背景

目标：构建一个Web应用，用户输入品类关键词，自动挖掘亚马逊Top Creator红人并展示。

原始需求演变：
1. 最初：按品类搜索 → 爬产品 → 提取红人（成本高，技术复杂）
2. 改进：用Google Search API获取红人名单 → 验证Top Creator（相对简单）
3. 最优解：直接爬Amazon Live按品类浏览的创作者 → 批量验证 → 展示结果

## 最终方案：Amazon Live 爬虫方案

### 核心思路
```
用户输入品类 (如"Electronics")
    ↓
Actor 1: amazon-live-creators-scraper
  - 访问 amazon.com/live/browse/[category]
  - 提取该品类的所有 Featured Creators 的 username

    ↓
Actor 2: amazon-influencers-profile-scraper (已存在)
  - 批量输入 username 列表
  - 返回：username, display_name, bio, top_creator_status, posts_count 等

    ↓
后端处理 + 前端展示
  - 表格展示结果
  - 支持排序、筛选、导出
```

### 优势
✅ 数据来自官方Amazon Live（最权威）
✅ 完全自动化（品类输入 → 结果输出）
✅ 成本可控（初始$20-30，每月$5-10维护）
✅ 可靠性高（Amazon Live更新频率低）
✅ 易于扩展（支持所有品类）

### 成本预估
- Amazon Live爬虫Actor运行：$20-50/月（取决于频率和数据量）
- Apify验证Actor：$5-10/月（按结果数计费，$5/1000）
- 总成本：$25-60/月

### 可靠性
- 爬虫可靠性：85-90%（Amazon有反爬虫机制，但Amazon Live更新频率低，相对稳定）
- Top Creator验证：95%（Apify官方工具，经过测试）

## 实现计划

### Phase 1: 创建 Amazon Live 爬虫 Actor
- 文件：actors/amazon-live-creators-scraper/
- 语言：JavaScript + Apify SDK
- 功能：
  * 接受品类名作为输入
  * 爬取 amazon.com/live/browse/[category]
  * 提取创作者信息（username, name, bio 等）
  * 输出到 Dataset

### Phase 2: 创建 Web 应用后端
- 技术栈：Node.js + Express
- 功能：
  * 接收品类输入
  * 调用 Actor 1 获取创作者列表
  * 调用 Actor 2 验证 Top Creator
  * 数据合并、清洗、排序
  * 返回 JSON 结果

### Phase 3: 创建 Web 应用前端
- 技术栈：React（或简单的HTML+JS）
- 功能：
  * 输入框 + 搜索按钮
  * Loading 状态显示
  * 结果表格展示（含排序、筛选）
  * 导出 CSV
  * 复制按钮

### Phase 4: Cloudflare 部署
- 后端：Cloudflare Workers
- 前端：Cloudflare Pages
- 环境变量管理：.env (APIFY_API_TOKEN)

## 技术细节

### Amazon Live 品类 URL 结构
- amazon.com/live/browse/featured（所有Featured Creators）
- amazon.com/live/browse/[category-slug]（按品类）
- 已验证可浏览品类：Fashion, Beauty, Electronics, Fitness, Food, Home等

### Apify Actor 集成
- Actor 1 输出格式：
  ```json
  {
    "username": "creator_name",
    "displayName": "Display Name",
    "bio": "Bio text",
    "profileUrl": "https://amazon.com/shop/creator_name"
  }
  ```

- Actor 2 输入格式：
  ```json
  {
    "influencer_names": ["creator1", "creator2"],
    "country": "US"
  }
  ```

- Actor 2 输出格式：
  ```json
  {
    "username": "creator_name",
    "displayName": "Display Name",
    "bio": "Bio",
    "topCreatorStatus": true,
    "postsCount": 245
  }
  ```

```bash
cd backend
npm install
npm start
```

### 3️⃣ 打开前端
在浏览器打开：`frontend/index.html`

### 4️⃣ 开始搜索
选择品类 → 点击搜索 → 查看结果
